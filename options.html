SCMS Options

The easyest way to get a full list of options is to use the built in help


   $ scms --help

Working Directory

By defualt scms processes files in the current working directory. This can by changed by specifing a source


  $  scms --source /pathto/www.mywebsite.com

Output Directory

By defualt scms compiles files in the current working directory. This can by changed by specifing a destination


  $  scms --destination /pathto/outputdir

Running multiple sites

Occasionally you may want to run multiple scms sites in serve mode. This will cause a port clash for the web servers, but you can avoid this by specifying a port yourself (default = 8002).


  $  scms --server --port 8003

Live reload

scms can be used with http://livereload.com/.

You can either use the browser plugins or add <%= page.livereload %> to your layout template
