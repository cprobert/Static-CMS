Easyily Build Super-Fast Static Websites!

A ruby gem for creating simple static websites in minutes.


  $ gem install scms
  $ scms --create mynewsite
  $ cd mynewsite
  $ scms --serve

scms also comes with a WYSIWYG editor called Air-Monkey, so your clients can edit their sites in point and click environment.

check the prerequisites

Built in web server

Use the --serve option to build and view your website


  $ scms --serve

To stop the built in web server use [Ctrl] + [C]

Change dectection

The --watch option to detect changes to your views and layouts:


  $ scms --watch

To stop the filewatcher use [Ctrl] + [C]

Multi options

You can use multiple options together


  $ scms --watch --serve

Note: scms processes files in the current working directory
