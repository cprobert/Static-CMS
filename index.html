Easyily Build Super-Fast Static Websites!


  $ gem install s3cms

check the prerequisites

Creating a new site

To create new website:


  $ s3cms -a create -w pathto\www.mynewsite.com

The new html files will be found in the "websites" folder.

Compile

To compile the templates into a html website:


  $ s3cms -a build -w pathto\www.mynewsite.com

The default action is build(feel free to exclude the -a flag).

Publish

To publish your website:


  $ s3cms -a deploy -w pathto\www.mynewsite.com

The default website path is the current working dir.
