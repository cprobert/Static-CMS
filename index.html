Simple :)

Static-CMS is easy to use. I originally wrote this system to enable my children (9 & 10 years old) to build their websites.

Because Static-CMS uses templating, you can quickly develop a website with reusable templates and dynamic views (eruby/erb).

Get Started »

Fast :)

Many of the features of Static-CMS are focused on speed.

Including: The ability to merge all your javascripts and stylesheets files, minimising download requests.

On deployment all javascript and stylesheets are minimised (using YUI Compressor).

These measures dramatically speed up download time.

Money Saving :)

Static-CMS has Amazon S3 deployment built in. S3 is fantastic for fast inexpensive hosting of static websites.

Run the deploy script to publish your static website to an Amazon S3 bucket.
