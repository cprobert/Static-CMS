How to Install Static-CMS

check the prerequisites


	gem install scms

Creating a new site

Run "menu.cmd" to open the command menu. To compile the website:


	scms -a create -w pathto\www.mynewsite.com

The new html files will be found in the "websites" folder.

Compile

To compile the templates into a html website , type the following in to the command menu:


	scms -a build -w pathto\www.mynewsite.com

Publish

To your website, type the following in to the command menu:


	scms -a deploy -w pathto\www.mynewsite.com
