Sass

Sass (Syntactically Awesome Stylesheets) is a stylesheet language that extends css, adding additional functionality such as varibles, nesting and mixins. Checkout the Sass website to learn more.

To make life a bit easier, Static-CMS automatically parses all scss/sass files, and converts then to .css files.

Asset Bundling

It's best practise to minimise your download requests. Static-CMS has a system that allows you to 'bundle' your css and javascript files.

Create a file with a .bundle extention, and list the files you want bundled:


	_source/base.css
	_source/bootstrap.css
	_source/bootstrap-responsive.css

Whats going on here?

The bundler with enumerate the files in a .bundle file and concatanate them into a single file. Javascript files will be automatically minified, unless they end with a .min extention, e.g. myfile.min.js

You then replace your script and css references in your skin file like this:


    <link href='<%= page.bundles["css/styles"] %>' type='text/css' rel='stylesheet'/>
    <script src='<%= page.bundles["js/libraries"] %>'></script>
